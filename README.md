# Hotel Management System

A comprehensive and user-friendly Hotel Management System designed to streamline daily hotel operations. This application automates various management tasks such as room booking, customer record management, billing, and room service tracking, making hotel administration efficient and error-free.

## Features

- **Room Management:** Add, update, view, and delete room details (Room type, price, availability status).
- **Customer Check-In/Check-Out:** Seamless registration of guests, assigning rooms, and managing check-out processes.
- **Booking & Reservation:** Easily book rooms in advance and check real-time room availability.
- **Billing & Invoicing:** Automatically calculate total stay costs, food bills, and generate detailed invoices upon check-out.
- **Search & Filter:** Search for guest records or available rooms quickly using various filters.
- **Data Persistence:** Records are securely saved and retrieved.

## Tech Stack

- **Language:** Java(Swing) 
- **Database:** MySQL
- **Paradigm:** Object-Oriented Programming (OOP)
